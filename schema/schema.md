# Metadata Schema

## Name
BoardGame Interaction Schema

## Main entity
BoardGame

## Description
Metadata schema designed to describe board games as information resources, focusing on social interaction between players.

## Properties
- name (Schema.org)
- description (Schema.org)
- numberOfPlayers (Schema.org)
- playingTime (Schema.org)
- interactionType (local)
- socialDynamics (local)
