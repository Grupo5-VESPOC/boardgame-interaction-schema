# Application Profile

## Context of use
This application profile is intended for use in museums of play, ludotecas, educational environments, and personal board game collections.

## Users
- Educators
- Information professionals
- Players
- Game designers

## Metadata usage rules

| Property | Mandatory | Repeatable | Justification |
|--------|----------|------------|--------------|
| name | Yes | No | Basic identification |
| numberOfPlayers | Yes | No | Game selection |
| playingTime | Recommended | No | Educational context |
| interactionType | Yes | Yes | Core of the project |
| socialDynamics | Recommended | Yes | Social analysis |
